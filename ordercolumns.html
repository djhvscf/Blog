Order columns of a gridview when AutoGeneratedColumns is true

Hello! Again, yes again. I’m here to explain other headache of a programmer… How can I order columns when autogeneratedcolumns of a grid is true? So, Here is the solution: You have to set the onRowCreated event and add this function in your codebehind, that’s all.

C# code:
					protected void GridView1_onRowCreated (object sender, GridViewRowEventArgs e)
					{
					  GridViewRow row = e.Row;
					  List< TableCell > cells = new List< TableCell >();
					  foreach (DataControlField column in GridView1.Columns)
					  {
						//Getting first Column of the Gridview
						TableCell cell = row.Cells[0]; 
						//Remove that cell from the gridview
						row.Cells.Remove(cell);
						//Adding that cell as the last cell in the gridview
						cells.Add(cell);
					  }
					  // Add cells
					  row.Cells.AddRange(cells.ToArray());
					}
VB.NET code:
