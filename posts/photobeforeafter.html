Before and after effect

Hello! Again, yes again. I'm here to share with you a jQuery plugin about before/after effect. Here is the code:

Javascript code:
					$(".topImage").css('width', '50%');
					$(".beforeAfterSlidebar").mousemove(
						function(e) {
							// get the mouse x (horizontal) position and offset of the div
							var offset =  $(this).offset();
							var iTopWidth = (e.pageX - offset.left);
							// set width of bottomimage div
							$(this).find(".topImage").width(iTopWidth);
						}
					);
CSS code:
					.beforeAfterSlidebar {
					  position: relative;
					  width: 800px;
					  height: 600px;
					  cursor: pointer;
					}
					.beforeAfterSlidebar div {
					  position: absolute;
					  top: 0px;
					  left: 0px;
					  overflow: hidden;
					}
					.topImage {
					  border-right: solid 4px black;
					}
