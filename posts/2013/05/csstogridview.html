Applying css to gridview (aspx)

First the gridview is divided in rows, columns, header and footer. Each of these parts of the gridview can be have a css style. How? Just like that:

1. First, define these properties in the gridview:

					
					  //Header
					  // Row
					 //Row
					
					
					...
					
					
					  //Footer
					  //Footer

2. Define these rules in CSS:


					/*GRID*/
					.Grid
					{
					font-family:Arial;
					background-color: #FFFFFF;
					border: solid 1px #CCCCCC;
					}

					.Grid td {
					padding: 5px 5px 5px 5px;
					}

					.GridHeader
					{
					background-color: #9CC525;
					padding:50px 0px 50px 0px;
					}

					.GridHeader th
					{
					padding:5px;
					color: #FFFFFF;
					}

					.GridRow
					{
					color: #000000;
					background-color: #FFFFFF;
					}
					/*END GRID*/
