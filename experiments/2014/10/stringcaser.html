StringCaser a jQuery plugin

This is my new jQuery plugin that convert a string separated string to CamelCase: foo-bar → fooBar or underscore_case: foo-bar → foo_bar. You can see and download the code from GitHub.

Now I'm going to teach you how to use it.

You have to pass the DOM element and if the elements is input or textarea the plugin will bind paste, blur, change events. The plugin have two parameters: 1- method: Means CamelCase or underscroe_case and 2- humanReadable: Means that the conversion of the string is human readable

Javascript code:
						$('#input').stringcaser({method:'CamelCase', humanReadable: true});
						$('#textarea').stringcaser({method:'underscore_case', humanReadable: false});
						$('#span').stringcaser();
