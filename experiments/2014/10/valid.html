jValid a jQuery plugin

This is my new jQuery plugin to make valid inputs on form fields and HTML elements. You can see and download the code from GitHub.

Now I'm going to teach you how use it.

You have to pass the regex with 0 when you want a number and S when you want a letter. Or if you want use a regular expression you have to pass the regular expression and behaviorRegExp: true

Javascript code:
						$('#inputmasked').jValid({regex:'0-0000-0000'});
						$('#crid').jValid({regex:'0-0000-0000'});
						$('#date').jValid({regex:'00/00/0000'});
						$('#datehour').jValid({regex:'00/00/0000 00:00:00'});
						$('#telephone').jValid({regex:'0000-00-00'});
						$('#telephonecode').jValid({regex:'(000) 0000-00-00'});
						$('#zip').jValid({regex:'00000-000'});
						
						//Regular expressions
						
						$('#regnumbers').jValid({regex:'[0-9]', behaviorRegExp: true});
						$('#regletters').jValid({regex:'[a-zA-Z]', behaviorRegExp: true});
						$('#regcrazy').jValid({regex:'[a-dD-Z]', behaviorRegExp: true});
