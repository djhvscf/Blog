flatJSON a jQuery plugin

This is my new jQuery plugin that convert a JSON object to flat JSON object You can see and download the code from GitHub.

Now I'm going to teach you how to use it.

You have to pass a JSON object to plugin calling the main function, like:

Javascript code:
						//Create the object
						var objTest = 
									[
										{
											test1: {
														extra:'1'
													}, 
											test2: {
														test3: '3'
													}, 
										    'test4': '4', 
											test5: {
														test6:{
																test7:'7', 
																test8:'8'
															   }, 
													   test9:'9'
												   }
									   }
								   ];
						//Call the functiion passing the object and then this function will return a flat JSON object
						$.flatJSON({data:objTest});

And then when We receive the object We can see the difference between objects
