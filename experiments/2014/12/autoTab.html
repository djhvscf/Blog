autoTab a jQuery plugin

This is my new jQuery plugin that allows auto tab between input and text areas elements. You can see and download the code from GitHub.

How to use it?

Define the property data-tab in the input element setting a number between 0 to N like data-tab="0" and declare data-length in the input element setting a number between 1 to N like data-length="1" this property will be the maxlength property in each input element and it helps to plugin knows when change the focus.
